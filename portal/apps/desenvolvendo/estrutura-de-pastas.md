# Estrutura de pastas de uma app

A estrutura de pasta de uma app é:

```
.
├── meta.json
└── storefront
    ├── assets
    ├── libs.json
    ├── models
    ├── pages
    ├── partials
    ├── routes.json
    ├── rules.json
    └── widgets
```

Para mais informações sobre cada arquivo e pasta, leia o capítulo sobre [recursos](../../recursos/).
